bbPager 3.7
description
bbPager is a VWM (Virtual Window Manager) plugin for Blackbox for Windows:
  • see window positions on all workspaces
  • change workspaces with the pager
  • focus windows on any workspace
  • move windows between workspaces

    • bbPager 3.7 requires any Blackbox for Windows flavour issued since 2005 & Windows 98 or higher.

    NC-17

    initial setup
  • Use the load plugin menus provided in your flavour of bb4win and browse to the location of the bbPager dll
  • OR add the relative path of the plugin to the BB configuration file plugins.rc
    • located in the Blackbox directory
    • OR accessed by Win+P with bbKeys)
    For example, if Blackbox is installed in c:\Blackbox, and bbPager.dll is located in c:\Blackbox\plugins\bbPager\, you should add the line...

    plugins\bbPager\bbPager.dll ...to plugins.rc.

    Simple, eh? :)

    how to use bbPager...

    Mouse Input

    Once the plugin is loaded, several things can be done with a simple click on the pager window:

    LeftClick + Ctrl  - anywhere in the Pager this lets you drag the Pager around
    LeftClick + Shift  - same as MiddleClick - when there's no 3-button mouse
    RightClick  - inside the current Desktop & outside all Desktops, access the plugin Menu
    RightClick + Ctrl  - access the plugin Menu anywhere in the Pager
    RightClick + Shift  - toggle in & out of the slit
    RightMouseButton + LeftClick  - Show the BroamsMenu
    MouseScroll  - change Desktops, when the pager has the focus

    FocusButton  - focus on target Window - default LeftClick
    FocusButton + Shift  - bring target Window to current Desktop (not xoblite)
    MoveButton  - focus on target Window & drag it around - default MiddleClick
    MoveButton + Shift  - drag non-target Window in the Pager
    ChangeButton  - change Desktops - default RightClick

    The Menu

    Most items should be self-explanatory - except perhaps the "Settings" submenu.

    Making any change via the plugin menus saves those settings to the RC file. "Reload Settings" allows manual changes to the RC file to take place directly.

    bbPager Styling is read in the following order:

  • style settings in bbpager.bb - Settings|Edit Style
  • bbPager settings in the style file
  • default style elements (windows are not always visible)

    • Keyboard Input

    Make bbPager the focused window (by clicking on empty space in the active desktop) & it can accept keyboard input. The keys you can press to access a few functions quickly are below. Note: to re-position you need NumLock on.

    Numpad Keys 1-9, except 5  Set bbPager position (7 is TopLeft, 3 BottomRight, etc.)
    Numpad Key 5 or Enter or Back  Toggle autoHide
    Numpad Add  Toggle alwaysOnTop
    Numpad Subtract  Toggle vertical|horizontal orientation
    Escape  This cancels window movement if a window has been grabbed in the pager

    Miscellaneous Notes

    Moving bbPager: When in autohide mode, dragging of the pager window is not possible - this is where the positioning menu comes in handy. You cannot drag the pager window off the virtual/viewable screen area manually either. To put the pager on a different monitor, Ctrl+LeftClick and drag the plugin into the prospective monitor's screen area - using the placement menu or placement keyboard shortcuts will now position the pager window in this monitor's screen area accordingly.

    Autohide: Enabling autohide when the pager window is not on a screen edge will cause the pager to snap to the nearest screen edge and enter the hidden state.

    Moving windows:

  • When an active window is moved, the workspace is changed to the one it is moved to.
  • Holding down the Shift key can prevent the changing of workspaces - as long as the target workspace is not empty.
  • When unfocused windows are moved the workspace does not follow the window.
  • Maximised windows cannot be moved within the same workspace.
  • When they are moved to another workspace they will stay in their maximised state.
  • Maximised windows use the cursor position to decide which desktop it is dragged to, while unmaximised windows use upper left corner of the window to decide this.

    • bbPager.rc settings

    Default settings are:

    bbPager.x: 895  X position of the pager
    bbPager.y: 10  Y position of the pager
    bbPager.sizeRatio: 20  Number of times screen is scaled down for bbPager
    bbPager.alignment: horizontal  Chooses whether rows or columns are used
    bbPager.columns: 1  Number of columns for desktops in vertical mode
    bbPager.rows: 1  Number of rows for desktops in horizontal mode
    bbPager.desktopChangeButton: 2  Mouse button for changing desktops
    bbPager.windowMoveButton: 3  Mouse button for moving windows
    bbPager.windowFocusButton: 1  Mouse button for focusing windows
    bbPager.onTop: true  Always on top?
    bbPager.autoHide: false  Toggle autohide mode for the pager
    bbPager.snap: 20  Distance before window locks to screen edge
    bbPager.desktopNumbers: true  Show numbers of desktops?
    bbPager.fullScreen: false  Position according to multi-monitors?
    bbPager.useSlit: false  Hold the pager inside the Slit?

    bbPager.alphaValue:   Value of transparency (0-8) -> default is Menu.alpha

    Mouse buttons: For setting mouse buttons to certain actions, 1 is the Left Mouse Button, 2 is the Right Mouse Button, and 3 is the Middle Mouse Button.

    bbPager bro@ms

    @BBShowPlugins  Standard BB bro@m to show all plugins
    @BBHidePlugins  Standard BB bro@m to hide all plugins

    @bbPager.Reload  Make bbPager reload its settings
    @bbPager.OpenRC  Open the bbpager.rc file with your preferred editor
    @bbPager.OpenStyle  Open the bbpager.bb file with your preferred editor
    @bbPager.LoadDocs  Load the documentation file in your browser
    @bbPager.Vertical  Change the alignment to vertical mode
    @bbPager.Horizontal  Change the alignment to horizontal mode
    @bbPager.About  Display message box about the plugin

    Value & True|False bro@ms can also be set for most of the bbPager entries:
    @bbPager.snap   @bbPager.onTop   @bbPager.autoHide   @bbPager.alphaValue   @bbPager.sizeRatio   @bbPager.desktopNumbers   @bbPager.togglePlugins   @bbPager.useSlit   @bbPager.rows   @bbPager.columns  

    POSITIONING: These set the position of the pager window to that specified, according to bbPager.fullScreen (false = current monitor only; true = full screen)
    @bbPager.TopLeft   @bbPager.TopCenter   @bbPager.TopRight   @bbPager.CenterLeft   @bbPager.CenterRight   @bbPager.BottomLeft   @bbPager.BottomCenter   @bbPager.BottomRight
